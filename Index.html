GITHUB

Aplicacion para guardar proyectos y poder realizar cambios desde cualquier dispositivo con GITHUB DESKTOP

Pasos para crear repositorio:

  • Crear usuario/clave en pagina de Github
  • Crear nuevo archivo que se guarde en Disco Local C/XAMPP/HTDOCS/EMPRESA_A
  • Abrir desde el Github y clonar el archivo para seguir con su edicion
